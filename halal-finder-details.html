Overview

HalalFinder is an innovative web application designed to serve the Muslim community by providing easy access to halal restaurants, mosques, and community events. The platform combines multiple essential features into one convenient location, making it easier for Muslims to find what they need in their area.

Key Features

  • Interactive map showing nearby halal restaurants and mosques
  • Prayer time calculator based on user location
  • Qibla direction finder with compass integration
  • Community event calendar and notifications
  • Restaurant reviews and ratings from the Muslim community
  • Mosque prayer schedule integration

Technologies Used

  • HTML5 & CSS3 for responsive frontend design
  • JavaScript for interactive map features
  • Firebase for real-time data and user authentication
  • Google Maps API for location services
  • Prayer Times API for accurate prayer calculations
